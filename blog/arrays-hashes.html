1.3:Arrays and Hashes

Can you explain the difference between Arrays and Hashes?
We're going to be discussing the difference between Arrays and Hashes, providing a few examples along the way and the use for each!We first need to establish the fact that both are collections used to store data and are very similar. Arrays, are indexed collections that can contain strings(text), integers and even other Arrays! They are indexed by integers, so for example, assuming you have an array that looks like this:
array=["Hi",25,"name",[24,30]]
Here we say our Array contains strings, integers and even another Array. If you wanted to access the first item in the array we would do so by calling array[0](The index starts at 0). This is particularly helpful in the case of a grocery list program. Assuming you have ten items and you'd like to know the first item that you need to purchase you can do so with the example we used.
Let's move on and talk a bit about Hashes now. Hashes are similar to Arrays as they are collections used to store data as well, however they differentiate in structure. Hashes consist of a key/value pair ( as opposed to the integer indexation of an Array) and keys are unique in Hashes.Here's what a Hash looks like:
hash={name:"Sabri",address:"1234 jasonbourne st"}
We clearly see the key(eg:name) and its value pair("Sabri"). The key has to be unique however we can have the same value several times. When we want to find the value of a specific key we do as such hash[:name] and that should return "Sabri" in this case.
Now we know what an Array is and what is Hash and their differences but why use one over the other ? That really depends on the nature of your program and what you are trying to accomplish. We know that both are collections and indexed differently but if you're trying to accomplish a certain task choosing one over the other can make things a little bit easier. For example, if you're building a collection of phone numbers of friends, using names as a key and returning the phone number as a value is a lot easier than an integer index. This is just one of the examples of uses of a Hash over an Array and it really depends!
